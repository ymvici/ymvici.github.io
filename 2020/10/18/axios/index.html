axios

axios是基于promise对ajax的一种封装

基本使用

axios({
     method: 'get',
     url: 'http://elm.cangdu.org/v1/cities?type=hot',
}).then(res => this.hotcity = res.data) 

axios.get('http://elm.cangdu.org/v1/cities?type=hot')
     .then(res => this.hotcity = res.data)

携带参数的get请求

axios({
     method: 'get',
     url: 'http://elm.cangdu.org/v1/cities?type=hot?id=1',
}).then(res => this.hotcity = res.data)

或者

axios({
     method: 'get',
     url: 'http://elm.cangdu.org/v1/cities?type=hot',
     params: {
         id: '1'
     }
}).then(res => this.hotcity = res.data) 

axios.get('http://elm.cangdu.org/v1/cities?type=hot',
     {params:{id:'1'}}
     ).then(res => this.hotcity = res.data)

post方法

axios({
     method: 'post',
     url: 'http://elm.cangdu.org/v1/cities?type=hot',
     data: {
         id: '1'
     }
}).then(res => this.hotcity = res.data)

axios的并发请求

第一种方法，返回来的是一个数组

axios.all([
    axios.get('http://elm.cangdu.org/v1/cities?type=hot'),
    axios.get('http://elm.cangdu.org/v1/cities?type=city',{params:{id:'1'}})
    ]).then(res => this.hotcity = res.data
     ).catch( err=> console.log(err))

第二种方法

axios.all([
    axios.get('http://elm.cangdu.org/v1/cities?type=hot'),
    axios.get('http://elm.cangdu.org/v1/cities?type=city',{params:{id:'1'}})
    ]).then(
        axios.spread((res1,res2) => {
            console.log(res1)
            console.log(res2)
        })
     ).catch( err=> console.log(err))

axios的全局配置

axios.defaults.baseURL = 'http://elm.cangdu.org/v1'
axios.defaults.timeout = 5000
axios.get('cities?type=hot').then(res => {
    console.log(res)
})

axios的实例

let newAxios = axios.create({
    baseURL:'http://elm.cangdu.org/v1'
    axios.defaults.timeout = 5000
});

axios的拦截器

axios提供了两大类拦截器，一种是请求方向的拦截（成功请求，失败请求）；另一种是响应方的（成功请求，失败请求）

作用：用于网络请求在发起时或响应时对操作做出的响应处理

发起请求时可以添加网页加载动画，使用token认证时检查是否携带token信息
响应时可以进行对数据进行相应的处理

请求方向

客户端发送请求 –> [请求拦截器] –> 服务器

axios.interceptors.request.use(config =>{ // config是发请求时携带的数据
    // 携带token
    let token = localStorage.getItem('token')
    token && (config.headers.Authorization = token) // 如果token存在则执行后面的语句
    return config
},error => {
    return Promise.reject(error)
})

axios.get('http://elm.cangdu.org/v1/cities?type=hot',
     {params:{id:'1'}}
     ).then(res => this.hotcity = res.data)

响应方向

axios.interceptors.response.use(config=> {
    console.log('进入响应拦截器')
    console.log(config)
    return config // 请求得到数据res对象
}).catch(err=>{
    console.log('响应方向失败')
    console.log(err)
})

axios.get('http://elm.cangdu.org/v1/cities?type=hot',
     {params:{id:'1'}}
     ).then(res => this.hotcity = res.data)

axios封装

初级封装

import axios from 'axios'
export function request(config, success, fail) {
    axios({
        url: config
    }).then(res => {
        success(res)
    }).catch(err=> {
        fail(err)
    })
}

调用 
import {request} from './network/request/request'
request('http://elm.cangdu.org/v1/cities?type=hot',res=> {
    console.log(res)
},err=> {
    console.log(err)
})

封装到对象中

import axios from 'axios'
export function request(config) {
    axios.default.baseURL = 'http://elm.cangdu.org/v1'
    axios(url: config.url).then(res => {
        config.success(res)
    }).catch(err=> {
        config.fail(err)
    })
}

调用
import {request} from './network/request/request'
request({
    url: 'cities?type=hot',
    success:res => {
        console.log(res)
    },
    fail:err => {
        console.log(res)
    }
})

qs库

什么是qs?

qs是一个流行的查询参数序列化和解析库。可以将一个普通的object序列化成一个查询字符串，或者反过来将一个查询字符串解析成一个object,帮助我们查询字符串解析和序列化字符串。

vue中qs安装使用

  • 步骤：
    • 打开控制台输入：npm install qs
    • 在main.js中导入qs插件：import qs from 'qs'
    • 在main.js中配置全局属性：Vue.prototype.$qs = qs

方法：qs.parse()和qs.stringify()

  • qs.parse()是将URL解析成对象的形式

    const str = "username='admin'&password='admin'" 

console.log(qs.parse(str)) // Object { username: "admin", password: "admin" }

  • qs.stringify()是将对象 序列化成URL的形式，以&进行拼接

    qs.stringify({username:'admin', password:'admin'})

//username=admin&password=admin
